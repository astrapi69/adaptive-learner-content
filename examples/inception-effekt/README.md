# Example lesson — Der Inception-Effekt

A complete, real lesson you can read end-to-end as a reference.
[`lesson.json`](lesson.json) is a verbatim copy of the shipped lesson:

- **Live in the catalogue as:** `sets/de/psych-intro/lessons/106-inception-effekt-und-beeinflussung.json`
- **Domain:** `psychology` (non-language; `source_language == target_language == de`)
- **Shape:** 10 cards · 3 theory sections · 8 exercises
  (3 × matching, 2 × picture_choice, 2 × free_text, 1 × word_tiles)

It demonstrates the patterns a good knowledge lesson uses:

- theory → exercise interleaving (receptive flow),
- every `free_text` has ≥ 2 accepts **and** distractors,
- every `matching` has ≥ 3 pairs,
- `picture_choice` uses the text-label convention with one `is_correct: "true"`,
- real umlauts and a sachlich, descriptive framing for an influence topic.
